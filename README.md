# Hareem Khan — Personal Portfolio

A personal portfolio website built with React, showcasing projects, work experience, awards, and a contact form. Designed with a warm, professional aesthetic using a neutral/cream color palette.

🌐 **Live Site:** [hareemk.com](https://hareemk.com)

---

## Features

- **Home** — Hero section with background video, an about me section, and featured project cards
- **Projects** — Full project grid with featured projects, image galleries, tech stack tags, and detail pages for each project
- **Experience** — Slideshow-based work history and awards/credentials carousel
- **Resume** — Embedded resume preview with a direct download link
- **Contact** — Two-column layout with a live contact form (EmailJS) and social links

---

## Tech Stack

- **React** (Create React App)
- **React Router** — client-side routing
- **EmailJS** — contact form email delivery
- **react-slick** — experience and awards slideshows
- **Font Awesome** — icons
- **CSS** — custom styling with CSS variables; no UI framework
